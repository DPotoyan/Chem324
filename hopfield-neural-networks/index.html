Application: Hopfield Neural Networks

What you will learn

  • A Hopfield network as linear algebra in action. Neurons are a state vector s\mathbf{s}s, connections are a weight matrix WWW, and updates are matrix-vector products.

  • The update rule. Each neuron flips its state based on the sign of the weighted sum of its inputs, hi=(Ws)ih_i = (W\mathbf{s})_ihi​=(Ws)i​.

  • Energy minimization. The network evolves to minimize an energy function E=−12sTWsE = -\tfrac{1}{2}\mathbf{s}^T W \mathbf{s}E=−21​sTWs, settling into a stored pattern.

  • Associative memory. This is how the network recalls a complete pattern from noisy or partial input.

A Hopfield network is a type of recurrent neural network where all neurons are connected to each other. It is commonly used for associative memory: storing and recalling patterns. The network evolves to a stable state, recalling a stored pattern when given noisy or incomplete input. We can understand its behavior using matrix and vector operations.

Components of a Hopfield network¶

  1. Neurons (States): Each neuron has a binary state, either 1 (active) or -1 (inactive). The state of all neurons is represented by a vector s\mathbf{s}s. For a network of 3 neurons, the state vector might be:

    s=(1−11)\mathbf{s} = \begin{pmatrix} 1 \\ -1 \\ 1 \end{pmatrix}s=⎝⎛​1−11​⎠⎞​
    (1)

  2. Weights (Connections): Neurons are connected by weights, represented by a symmetric matrix WWW. For a 3-neuron network, the weight matrix might look like:

    W=(0w12w13w210w23w31w320)W = \begin{pmatrix} 0 & w_{12} & w_{13} \\ w_{21} & 0 & w_{23} \\ w_{31} & w_{32} & 0 \end{pmatrix}W=⎝⎛​0w21​w31​​w12​0w32​​w13​w23​0​⎠⎞​
    (2)

    The diagonal elements are zero because a neuron does not connect to itself.

  3. Update Rule: To update the state of neuron iii, we calculate the weighted sum of inputs from all other neurons:

    hi=∑jwijsj=(Ws)ih_i = \sum_{j} w_{ij} s_j = (W \mathbf{s})_ihi​=j∑​wij​sj​=(Ws)i​
    (3)

    If hi>0h_i > 0hi​>0, the neuron becomes active (si=1s_i = 1si​=1). If hi<0h_i < 0hi​<0, the neuron becomes inactive (si=−1s_i = -1si​=−1).

Example: A simple Hopfield network¶

Let us consider a 3-neuron Hopfield network with the following weight matrix:

W=(01−1101−110)W = \begin{pmatrix} 0 & 1 & -1 \\ 1 & 0 & 1 \\ -1 & 1 & 0 \end{pmatrix}W=⎝⎛​01−1​101​−110​⎠⎞​
(4)

Initial State

Assume the initial state of the network is:

s=(1−11)\mathbf{s} = \begin{pmatrix} 1 \\ -1 \\ 1 \end{pmatrix}s=⎝⎛​1−11​⎠⎞​
(5)

Update Neurons

We update each neuron based on the weighted inputs:

  1. Neuron 1: The input to neuron 1 is:

    h1=W11s1+W12s2+W13s3=0⋅1+1⋅(−1)+(−1)⋅1=−2h_1 = W_{11}s_1 + W_{12}s_2 + W_{13}s_3 = 0 \cdot 1 + 1 \cdot (-1) + (-1) \cdot 1 = -2h1​=W11​s1​+W12​s2​+W13​s3​=0⋅1+1⋅(−1)+(−1)⋅1=−2
    (6)

    Since h1<0h_1 < 0h1​<0, the new state of neuron 1 is s1=−1s_1 = -1s1​=−1.

  2. Neuron 2: The input to neuron 2 is:

    h2=W21s1+W22s2+W23s3=1⋅1+0⋅(−1)+1⋅1=2h_2 = W_{21}s_1 + W_{22}s_2 + W_{23}s_3 = 1 \cdot 1 + 0 \cdot (-1) + 1 \cdot 1 = 2h2​=W21​s1​+W22​s2​+W23​s3​=1⋅1+0⋅(−1)+1⋅1=2
    (7)

    Since h2>0h_2 > 0h2​>0, the new state of neuron 2 is s2=1s_2 = 1s2​=1.

  3. Neuron 3: The input to neuron 3 is:

    h3=W31s1+W32s2+W33s3=(−1)⋅1+1⋅(−1)+0⋅1=−2h_3 = W_{31}s_1 + W_{32}s_2 + W_{33}s_3 = (-1) \cdot 1 + 1 \cdot (-1) + 0 \cdot 1 = -2h3​=W31​s1​+W32​s2​+W33​s3​=(−1)⋅1+1⋅(−1)+0⋅1=−2
    (8)

    Since h3<0h_3 < 0h3​<0, the new state of neuron 3 is s3=−1s_3 = -1s3​=−1.

New State

After one round of updates, the new state of the network is:

s=(−11−1)\mathbf{s} = \begin{pmatrix} -1 \\ 1 \\ -1 \end{pmatrix}s=⎝⎛​−11−1​⎠⎞​
(9)

Storing patterns and energy¶

A Hopfield network stores specific patterns by adjusting the weights, and it evolves to minimize its energy. The energy of the network is defined as:

E=−12∑i,jwijsisj=−12sTWsE = -\frac{1}{2} \sum_{i,j} w_{ij} s_i s_j = -\frac{1}{2} \mathbf{s}^T W \mathbf{s}E=−21​i,j∑​wij​si​sj​=−21​sTWs
(10)

As the network updates, it reduces its energy until it reaches a stable state, which corresponds to one of the stored patterns.

Summary¶

  • Neurons are represented by a state vector s\mathbf{s}s.

  • Weights between neurons are represented by a matrix WWW.

  • Neurons update their states based on the weighted sum of inputs, which is a matrix-vector multiplication.

  • The network evolves to stable patterns by minimizing its energy function.

This is a basic introduction to the Hopfield network using matrix and vector operations.
